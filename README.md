# TodoApp

### NodeJS web application for creating to-do lists.

#### Link:
[ToDoApp](https://frozen-ravine-47059.herokuapp.com/)

#### Technologies:
* **Back-end** : Node.js (express, mongoose, passport), MongoDB.
* **Front-end**: HTML, CSS, JS, jQuery, bootstrap.
